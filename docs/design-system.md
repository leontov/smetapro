# Дизайн-система QuickEstimate Builder

## Бренд и айдентика
- **Тональность**: уверенная, технологичная, с акцентом на скорость и прозрачность.
- **Цветовая палитра**:
  - Основной фон — `#0F172A` (Slate 900) для тёмной темы и `#F8FAFC` (Slate 50) для светлой.
  - Акцент «действие» — `#F97316` (Amber 500): ключевые кнопки, индикаторы прогресса.
  - Акцент «аналитика» — `#14B8A6` (Teal 500): графики, подсказки ИИ.
  - Статусы: успех `#22C55E`, предупреждение `#FACC15`, ошибка `#EF4444`.
  - Нейтральные градации — оттенки Slate 100–700.
- **Типографика**:
  - Основной шрифт — Inter (fallback: SF Pro Text на iOS).
  - Размеры: заголовок H1 — 28pt/32px, H2 — 24px, body — 16px, caption — 13px.
  - Межстрочный интервал 1.4 для удобочитаемости на мобильном.
- **Иконография**: набор Phosphor, стиль `duotone`, stroke 1.5px.

## Компоненты
### Карточки
- Скругления 16px, тени `0 12px 24px -16px rgba(15, 23, 42, 0.6)`.
- Внутренние отступы 16px, gap 12px.
- Поддержка компактного режима (12px padding) для длинных списков.

### Sticky Action Bar
- Закреплена внизу экрана, учитывает `env(safe-area-inset-bottom)`.
- Содержит до трёх основных CTA (Сохранить, Актуализировать, Поделиться).
- На iOS добавляется виброотклик через `navigator.vibrate`.

### Табличные списки (Estimate items)
- Реализованы через карточки-строки, поддерживают swipe-жесты:
  - вправо — отметка как выполненной/актуализированной;
  - влево — доступ к быстрым действиям (редактировать, дублировать, удалить).
- Виртуализация списков для производительности.

### Формы и ввод
- Используются компоненты с плавающими метками, масками для чисел и валют.
- Валидация в реальном времени, статусы подсвечиваются цветами палитры.
- Для выбора единиц измерения — селектор с быстрым поиском.

### Диаграммы
- Библиотека Recharts с кастомной палитрой.
- Поддерживаются: столбчатые диаграммы для сравнения смет, линейные для динамики цен, пай-чарты по категориям.
- Визуализации имеют режим «ограниченный» для офлайна (без внешних данных).

### Пустые состояния
- Иллюстрации в стиле outline, краткий текст и CTA «Создать смету» или «Импортировать».
- Ссылки на раздел справки для погружения.

## Темы
- Светлая и тёмная темы; переключатель в настройках и автоопределение по `prefers-color-scheme`.
- Высококонтрастная тема: увеличенные цвета статусов, подчёркнутые границы, размеры шрифта +2px.

## Анимации и микроинтеракции
- Длительность 200–250 мс, кривая `cubic-bezier(0.4, 0, 0.2, 1)`.
- Skeleton-загрузчики для списков и карточек.
- Хаптика при подтверждении действий (если устройство поддерживает).
- Анимации отключаются при `prefers-reduced-motion`.

## Доступность
- Контрастность не ниже WCAG AA.
- Фокус-стили (обводка 2px акцентным цветом) всегда видимы.
- Размер интерактивных областей ≥ 44x44px.
- Поддержка VoiceOver: aria-label для кнопок, описания графиков.

## Документация и инструменты
- Библиотека компонентов ведётся в Storybook с вьюпортом iPhone 15.
- Дизайн-токены синхронизируются через Figma Tokens → JSON → Tailwind конфиг.
- Чек-лист визуального QA включает Lighthouse, Percy скриншоты и UX-тесты.
